# SimpleNotes
A Notes app that fetches data using Room database, allowing users to add, delete and search notes.

## Tech stack
* MVVM
* Jetpack compose
* Navigation
* StateFlow
* ROOM
* Hilt
* Coroutine

## Features
* Light/Dark theme
* Add note
* Delete single/multiple note
* Search note
* Staggered grid view
